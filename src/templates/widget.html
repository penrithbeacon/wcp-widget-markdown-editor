Agent Required

The Markdown Editor requires the Markdown Editor Agent — a small companion app that runs in the background and gives the widget access to your Mac's file system.

Install the agent once, and it starts automatically at login. Your files stay on your Mac and are never sent anywhere.

After installing, click Retry.
